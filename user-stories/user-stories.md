This file contains the core user stories derived from the use case diagram. These stories describe the main interactions users will have with the system.

---

## 1. User Registration

**As a user**,  
I want to be able to register an account,  
**so that I can list my properties and manage bookings.**

---

## 2. Property Listing

**As a registered user**,  
I want to be able to list a property,  
**so that potential renters can view and book it.**

---

## 3. Booking a Property

**As a guest**,  
I want to be able to book a listed property,  
**so that I can reserve a place to stay for my trip.**

---

## 4. Making a Payment

**As a guest**,  
I want to securely make a payment for a booking,  
**so that I can confirm my reservation.**

---

## 5. Writing a Review

**As a guest**,  
I want to leave a review after my stay,  
**so that I can share my experience and help other users make informed decisions.**
